## Bloakchain-demo

#### part1: 基本原型

* 区块：Block 当前区块信息、时间戳、哈希值和前一个区块的哈希值
* 区块链：Blockchain 通过array与map实现的数据结构
* 创世块：GenesisBlock 第一个块

```
   1.进入part1目录
   2.执行命令行 go build
   3.生成part1文件，执行命令行
        windows:
            part1
        mac：
            ./part1
```
